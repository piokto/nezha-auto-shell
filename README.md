## 哪吒面板一键反向代理
### 方法1：下载后运行（推荐）

```language
wget -O nezha-nginx.sh https://raw.githubusercontent.com/piokto/nezha-auto-shell/refs/heads/main/auto-nginx.sh
chmod +x nezha-nginx.sh
./nezha-nginx.sh
```

### 方法2：一键运行
```language
bash <(curl -fsSL https://raw.githubusercontent.com/piokto/nezha-auto-shell/refs/heads/main/auto-nginx.sh)
```
